# duckduckgo-web
A DuckDuckGo API for the web in javascript

## Getting Started
Step 1: Include ddgweb.js
```html
<script src="https://cdn.rawgit.com/Allen-B1/duckduckgo-web/8a4d95f3/ddgweb.js"></script>
```

Step 2:
```js
ddg.result("DuckDuckGo").then(function(result) {
  alert(result.text);
});
```
